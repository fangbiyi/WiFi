Machine Learning Toolbox

This toolbox implements some of the commonly used methods in machine learning, including data clustering, pattern classification, feature selection and extraction, dimensionality reduction, learning to rank, and so on.

Features

The Machine Learning Toolbox includes tools that can:

  • Perform various methods of data clustering
  • Train various methods of pattern recognition or data classification
  • Visualize the dataset and the trained classifiers
  • Check the consistency of the dataset before analysis
  • Perform input selection and feature extraction
  • Include interactive demos
